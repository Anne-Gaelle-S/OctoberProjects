# OctoberProjects

Le but de cette app est d'afficher la liste des derniers projets d'October.

## Installation & Lancement

* `git clone https://github.com/Anne-Gaelle-S/OctoberProjects.git`
* `cd OctoberProjects`
* `npm install`
* `npm start`

### Lancer les tests

* `ember test`
* `ember test --server`
